### 입력지표 설명 
#### resultcode 응답메시지 코드
#### resultMsg 응답메시지 내용

#### dataType 데이터타입(JSON, XML)
#### baseDate 발표일자 == 요청날짜 ex 20221010
#### baseTime 발표시각 == 요청시각 ex 0500
#### category - TMP 자료구분문자 - ex. TMP 1시간 기온 
#### fcstDate 예보일자 
#### fcstTime 예보시각 
#### fcstValue 예보값 es. 13도 ==자료값==
#### nx 55 예보지점 x좌표
#### ny 127  예보지점 y좌표 (격자 위경도 참고)
